# Trading Bot

## Description
This repository contains an Alpaca paper trading bot for MNQ futures that uses TradingView webhooks. The bot is configured to trade with a default position size of 10 lots.

## Features
- Alpaca paper trading API integration
- TradingView webhook setup
- Configurable parameters
- Logging and monitoring

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/vadlamudiashok/trading-bot.git
   ```
2. Install the necessary dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Set up your Alpaca API keys.
4. Configure the TradingView webhook URL.

## Usage
1. Run the bot:
   ```bash
   python trading_bot.py
   ```
